# Nastran-MBDyn
<h2>概要</h2>
Nastran・MBDynの研究進捗管理リポジトリです。
<br>
基本はStagingブランチに最終進捗を残していきます。
<br>
Stagingからブランチを切って作業をする。
<br>
ブランチ名は「日付_変更作業(addなど)_内容」
<h2>git操作</h2>

| 操作 | VS Codeでの操作 | コマンドライン(ターミナル) |
| :--- | :--- | :--- |
| **リポジトリの複製** | コマンドパレット > `Git: Clone` | `git clone [URL]` |
| **現在の状態確認** | 左側「ソース管理」パネルを表示 | `git status` |
| **今いるブランチ確認** | 左下のステータスバーを確認 | `git branch` |
| **変更をステージ** | ファイル横の **[+]** をクリック | `git add [ファイル名]` |
| **変更を記録** | メッセージを入力して **[コミット]** | `git commit -m "メッセージ"` |
| **ブランチ作成・切替** | 左下ブランチ名クリック > **[+ 新しいブランチ]** | `git checkout -b [ブランチ名]` |
| **ブランチの移動** | 左下ブランチ名クリック > リストから選択 | `git checkout [ブランチ名]` |
| **GitHubへ送信** | **[変更の同期]** または **[ブランチの公開]** | `git push origin [ブランチ名]` |
| **GitHubから取得** | **[プル]** (三点リーダーメニュー内) | `git pull origin [ブランチ名]` |
